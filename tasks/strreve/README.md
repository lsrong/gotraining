## Reverse a string 反转字符串
### Task
Take a string and reverse it.  
反转给定的一个字符串.

For example, "asdf" becomes "fdsa".  
例如， "asdf" 反转为 "fdsa"  


### Extra credit
Preserve Unicode combining characters.  
保留 Unicode 组合字符。

For example, "as⃝df̅" becomes "f̅ds⃝a", not "̅fd⃝sa".  
例如, "as⃝df̅" 反转为 "f̅ds⃝a", 而不是 "̅fd⃝sa".
